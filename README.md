# 微信公众号文章获取工具

一键抓取任意微信公众号的全部文章 URL。

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置目标公众号

编辑 `config.json`，填入目标公众号名称（需精确匹配）：

```json
{
  "targets": [
    { "nickname": "公众号名称" }
  ]
}
```

或者运行时直接指定：`python crawler.py --nickname "公众号名称"`

### 3. 安装浏览器内核 (Playwright)

本程序使用 Playwright 自动化获取微信后台凭证，因此需要安装 Chromium 内核：

```bash
playwright install chromium
```

### 4. 运行程序获取凭证

直接运行程序，它会智能提示你是想自动扫码还是手工输入：

```bash
# 本地运行（推荐）：按回车键，自动弹出浏览器扫码
python crawler.py --nickname "目标公众号"

# 仅获取最新的 10 篇文章
python crawler.py --nickname "目标公众号" --max 10

# 云服务器运行（受限说明：微信对云厂商 IP 节点及无头浏览器风控极严，极易被强制下线拦截！因此必须在本地电脑先人工获取凭证，再交给云端运行。）
python crawler.py --nickname "目标公众号"

# 命令行直传凭证（适合定时任务）
python crawler.py --credentials '{"cookie":"xxx","token":"xxx"}'
```

**⚠️ 如何手动获取完整的 Cookie 和 Token？**
*(注意：旧版的“书签一键提取代码”因浏览器安全限制，无法读取核心的 HttpOnly 登录凭证，现已废弃，必须使用以下开发者工具提取法)*
1. 用电脑浏览器登录 [微信公众平台](https://mp.weixin.qq.com)。
2. 按 `F12` 打开开发者工具，切到 **Network (网络)** 标签页。
3. 刷新页面，随便点击左侧一个以 `cgi-bin/` 开头的网络请求（如 `home?t=...`）。
4. 从你点击的这条请求的 URL 地址里，提取出你要的 `token=数字`。
5. 往下看面板里，找到 **Request Headers (请求标头)**，找到 `Cookie: ` 字段，把后面一整串极长的字符完整复制下来。
6. 把它们拼成 `{"cookie":"刚刚那一大串","token":"那个数字"}` 的 JSON 字符串，粘贴到提示框回车即可运行。

凭证会自动保存在本地 `credentials.json`，下次运行在过期前无需重新扫码。

### 5. 自动产生全量结果

程序现已实现全自动连贯执行：提取完文章基础列表后，**会自动无缝进入第二阶段**，全自动将列表里所有文章的纯文本详情（Content）逐个下载下来。

最终同时带有链接和正文的获取结果会保存在 `output/` 目录下，文件名为 `xxx_full_时间戳.json`，其中包含了标题、URL、发布时间以及完整的正文全文内容。

## ⚠️ 核心使用受限说明与注意事项

为了能长久稳定使用本脚本，请务必仔细阅读以下受限说明：

1. **凭证不具备持久性（几小时内即失效）**：
   提取出的 `Cookie` 和 `Token` **绝不是永久的**！微信的安全机制决定了扫码登录态寿命极短，通常在 **几小时甚至几十分钟（如果你变更了IP）** 就会过期。一旦程序报错需要重新提取，说明凭证已自动失效，你必须马上跑去公众号后台重新扫或者重新提取，**世界上没有永久绕过扫码拿数据的方法**。
2. **云服务器自动化受限（封号风险）**：
   **请勿**在云服务器（如阿里云、腾讯云机房节点）等非常规地理位置 IP 上使用无头浏览器（Headless）尝试全自动登录，微信对机房 IP 风控极其严格，一旦被风控，该公众平台账号可能直接被封禁网页端登录权限。请老老实实地在本地家里取 Cookie，再拿上云端跑！
3. **抓不到点赞/阅读量等互动数据**：
   受限于只能获取网页文章列表页设计，本程序**只能完美获取“所有文章元数据列表”和“文章内部的纯文本（Content）”**。至于一篇文章的阅读数、点赞数、评论数等，因保护极严只能通过截获微信手机客户端的通讯下发参数实现，此类纯外部浏览器抓取程序是做不到的。
4. **频率限制**：
   单个账号请求抓取间隔默认 **3 秒**，若获取超过几千篇的庞大公众号，请在 `config.json` 中适当调高 `delay_seconds`。长期高频狂刷会导致 IP 被微信官方拦截 24 小时。
5. 仅用于个人学习研究，请尊重著作权。
